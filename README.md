# Inventory Order Management System

A full-stack Inventory & Order Management System built with FastAPI, React, PostgreSQL, and Docker.

## Features

- Product Management
- Customer Management
- Order Management
- Inventory Tracking
- Stock Validation
- Unique Product SKU Validation
- Unique Customer Email Validation
- Dockerized Deployment

## Tech Stack

- FastAPI
- React
- PostgreSQL
- Docker
- Docker Compose
